Frequently Asked Questions

A subforum is the original content or topic that a user shares with The Draft community. Subforums are created to initiate discussions, share information, ask questions, or simply express thoughts and ideas. It consists of a title, which is a brief description of the content, the content, which contains text, and a photo file uploader for any phtos a user would like to add. A subforum can be edited using the "Edit Subforum" button located near the bottom of the subforum submission.

General Subforums are topics submitted by users that are of a general variety and that do not relate to a specific company. Conversely, Company Subforums are topics submitted by users that relate to a specific company. For example, a topic like "What are good networking tips for an introvert?" would be submitted on the General Subforums page (also the homepage), while a question like "How should I prepare for a Software Engineering interview at Google?" would be submitted on Google's specific company page on the site. To access a company's page the user should click the "View Companies" tab on the nav bar and selector on the particular company they are interested in discussing.

A post is text that a user submits that engages with a subform's content. Posts are displayed in their own cards underneath a subforum thread and should relate to the content of the original subforum. Posts can be thought of as replies to a subforum. A post can be deleted using the "Delete Post" Button

A comment is text that the user submits that engages with another user's post about a subforum. Comments are located directly under the corresponding post and can be thought of as replies to a post. A comment can be deleted using the "Delete Comment"

Likes are used to engage with subforums and allow the user to show appreciation towards a user's subforum submission. To a like a subforum, the user can click the heart button located directly underneath a subforum. Similar to a site like Reddit, subforums with more likes are displayed near the top of the page so that users can see what's popular on The Draft lately.

Applying to 100s of Jobs and struggling to keep up? If so, logged-in users can add a Job Application to their user profile by clicking on the "profile" tab on the site's nav bar at the top. After that fill out the "Add Job Application" form, select the current status of the job application, and upload any relevant files like the resume used and the cover letter sent (must be a pdf). Once submitted the user can now track the stages of the job application process and add components, notes, or update the job application with any relevant new information.

A component belongs to a Job Application and it's functionally meant to add aditional staging across an job application cycle. The "Type" field can be thought of as the particular stage of an application a user is in (e.g. Behavioral Interview, Panel Round, etc.), the "Date" should be the date this stage occurs on, the "contact" is who the interview was with, and the "description" box is text that describe how it went.

A note can be added to a Job Application Component if the user has any additional info that they would like to save to the particular note. Things like next steps, action items, or brainstorming for a take-home assignment are good use cases for a note.
