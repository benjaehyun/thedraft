What is a Subforum?

A subforum is a topic and/or question a user would like to engage

What is a Reddit Post? A subforum is the original content or topic that a user shares with The Draft community. It consists of a title, which is a brief description of the content, the conten which can contains text, links, images, videos, or a combination of these. Posts are created to initiate discussions, share information, ask questions, or simply express thoughts and ideas.
